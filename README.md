# Indian Pincodes

Nodejs based application to scrape indian pincodes from [india tv site](https://www.indiatvnews.com/india-pin-codes/)

## Install
npm ci

This should install node modules
* node-fetch
* cheerio
* axios
* bluebird

# Usage
`node app.js`
